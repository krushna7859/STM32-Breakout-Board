# STM32-Breakout-Board

## Design includes:
1) STM32F042K6Tx Processor
2) 16MHz High frequency Oscillator
3) BOOT0 and RESET features
4) USB Type-B Micro power adaptor
5) Serial wire Debugger On-board
6) I2S Protocol
7) Status LED.
